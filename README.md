Face Detection and Classification using Siamese Networks

Built a face verification system using MTCNN for face detection and a Siamese Neural Network for similarity learning. Preprocessed facial images with OpenCV, generated positive and negative image pairs, and trained a metric-learning model to determine whether two faces belong to the same individual. Implemented multiple regularization techniques to improve generalization, achieving 71.84% accuracy on the validation set.
